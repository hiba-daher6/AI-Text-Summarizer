# AI Text Summarizer

### Description:
This project allows users to upload your articles and texts and automatically generate a summary using AI models.
![Upload Screen](screenshots/upload.png)
![Summary Result](screenshots/result.png)

### How to Run:
##### -Install required packages:
pip install -r requirements.txt
##### -Run the app:
streamlit run app2.py

### Tech Stack:
transformers, streamlit, PyPDF2

### Future Improvements:
maybe added a button to download the summary and splitting and improving long text.
